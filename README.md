### CONTROLE DE INVENTÁRIO<br><br>

A aplicação AgilStore CONTROLE DE INVENTÁRIO foi desenvolvida em JavaScript e Node.js.
Realiza o controle de itens mostrando nome, preço, categoria e estoque, podendo editar qualquer
item bastando informar o id.

| index | nome    | preco | id            | categoria        | estoque |
|-------|---------|-------|---------------|------------------|---------|
| 0     | Teclado | 25    | 1767898434299 | periferico       | 0       |
| 1     | Monitor | 200   | 1767899164308 | tela             | 5       |
| 2     | Fone    | 250   | 1767899314352 | fone de ouvido   | 10      |
| 3     | Mouse   | 15    | 1767903615992 | mouse            | 15      |



Obs: Necessita do **node.js** instalado. 
Para veficar se está instalado na sua maquina, abra o prompt de comando(CMD) e digite: node --version, 
deverá mostrar a versão do node. -> v14.17.0 <- 


## Utilização do sistema


1. Abra a pasta da aplicação no vscode ou outro editor de código, e no prompt de comando, digite: **node** + **o nome do arquivo js** que foi criado.
Ex: **node index.js** , será exibido o script da aplicação.

2. Abra a pasta da aplicação, clique em arquivo -> abrir o windows powerShell, abrirá o prompt já direcionado na sua pasta. Digite: **node index.js** , será exibido o script da aplicação.

3. Se tiver o git bash instalado, abra a pasta da aplicação, click com o botão direito do mouse -> git bash. Abrirá o prompt já direcionado na sua pasta. Digite: **node index.js** , será exibido o script da aplicação.
